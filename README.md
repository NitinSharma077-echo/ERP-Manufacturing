# ERP Management System

A full-stack ERP system built with Node.js and Vite.

## Features

- User authentication
- Inventory management
- Dashboard analytics
- Product management
- Order tracking

## Tech Stack

Backend:
- Node.js
- Express

Frontend:
- Vite
- JavaScript
- HTML/CSS
